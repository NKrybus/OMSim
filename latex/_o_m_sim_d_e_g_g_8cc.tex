\doxysubsection{Detailed Description}
This file defines the implementation of the \mbox{\hyperlink{class_d_e_g_g}{DEGG}} class, which is responsible for constructing the D-\/\+Egg detector geometry. The class is derived from the {\ttfamily \mbox{\hyperlink{classabc_detector_component}{abc\+Detector\+Component}}} base class. It creates the pressure vessel, the inner volume, and places the PMTs inside the logical volume of the gel.
