\doxysubsection{Detailed Description}
\mbox{\hyperlink{classabc_material_data}{abc\+Material\+Data}} class is derived from \mbox{\hyperlink{classabc_data_file}{abc\+Data\+File}} class. It is designed to manage the material data, such as refractive index and absorption length.

\begin{DoxyNote}{Note}
This class is abstract and cannot be instantiated directly. Instead, use one of the derived classes\+: \mbox{\hyperlink{class_refraction_and_absorption}{Refraction\+And\+Absorption}}, \mbox{\hyperlink{class_refraction_only}{Refraction\+Only}}, \mbox{\hyperlink{class_no_optics}{No\+Optics}}, or \mbox{\hyperlink{class_ice_cube_ice}{Ice\+Cube\+Ice}}.
\end{DoxyNote}
