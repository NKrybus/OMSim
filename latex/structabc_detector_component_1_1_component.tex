This struct represents a single detector component within an \mbox{\hyperlink{classabc_detector_component}{abc\+Detector\+Component}} instance.  




\doxysubsection{Detailed Description}
A \mbox{\hyperlink{structabc_detector_component_1_1_component}{Component}} is defined by several properties which specify its physical properties and location within the detector. These properties include\+:


\begin{DoxyItemize}
\item {\ttfamily VSolid}\+: A pointer to a Geant4 G4\+VSolid object that represents the physical volume of the component.
\item {\ttfamily VLogical}\+: A pointer to a Geant4 G4\+Logical\+Volume object that contains the material properties, sensitivity settings and visualization attributes of the component.
\item {\ttfamily Position}\+: A G4\+Three\+Vector object representing the position of the component with respect to the origin of its containing volume.
\item {\ttfamily Rotation}\+: A G4\+Rotation\+Matrix object representing the rotation of the component with respect to its initial orientation.
\item {\ttfamily Name}\+: A G4\+String object representing the unique name of the component. All operations available in \mbox{\hyperlink{classabc_detector_component}{abc\+Detector\+Component}} utilize this struct to perform changes to handle all components.
