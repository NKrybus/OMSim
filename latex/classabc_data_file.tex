\doxysubsection{Detailed Description}
Derived classes should implement specific functionality for reading and writing different types of data files. This class contains a pointer to a \mbox{\hyperlink{class_parameter_table}{Parameter\+Table}}, used for storing file data, and a string for the file name.

\begin{DoxyNote}{Note}
This class is not meant to be instantiated directly, but should be subclassed. Methods in this class are to be overridden in the subclass for specific behavior. 
\end{DoxyNote}
