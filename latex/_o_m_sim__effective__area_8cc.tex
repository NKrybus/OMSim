\doxysubsection{Detailed Description}
The effective area of a module is calculated simulating a plane wave from a certain direction. The photon generation is made with the module \mbox{\hyperlink{class_angular_scan}{Angular\+Scan}}, running the method run\+Single\+Angular\+Scan(phi, theta) once for each direction to be investigated. Important command line arguments\+:

\begin{DoxyParagraph}{--input-\/file (-\/i)}
The input file to be processed. This should be a valid file path.
\end{DoxyParagraph}
\begin{DoxyParagraph}{--output-\/file (-\/o)}
The output file to which the results should be written. This should be a valid file path.
\end{DoxyParagraph}
\begin{DoxyParagraph}{--verbose (-\/v)}
Enables verbose mode, providing detailed logging information.
\end{DoxyParagraph}
\begin{DoxyParagraph}{--threads (-\/t)}
Specifies the number of threads to use for processing. This should be an integer.
\end{DoxyParagraph}
\begin{DoxyParagraph}{--phi (-\/p)}
The phi angle for the plane wave simulation. This should be a float.
\end{DoxyParagraph}
\begin{DoxyParagraph}{--theta (-\/t)}
The theta angle for the plane wave simulation. This should be a float.
\end{DoxyParagraph}
\begin{DoxyParagraph}{--seed (-\/s)}
The seed for the random number generator. This should be an integer. 
\end{DoxyParagraph}
